# Frontend PCMS

Vite ReactJS scaffold cho PCMS, dat o cung cap voi backend `pcms`.

## Chay project

```bash
npm install
npm run dev
```

## Nguyen tac chia folder

- `src/modules`: chia theo tung backend microservice.
- `src/core`: code nen tang dung chung nhu api, auth, storage, error, hook, util.
- `src/shared`: UI/component/hook/layout dung lai giua nhieu module.
- `src/app`: bootstrap app, provider, route tong, layout tong.
- `src/config`: cau hinh env, route, permission, service map.

Xem chi tiet tai [docs/FRONTEND_STRUCTURE.md](docs/FRONTEND_STRUCTURE.md).
